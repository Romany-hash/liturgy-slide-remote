# Liturgy Slide Remote Control

Control PowerPoint slides from your Android phone during liturgy.
 
## ✅ Features
- No installation
- Works on Windows 7 and later versions
- Control slides from up to 10 meters away
- Big buttons for easy tapping

## 📥 Download
[Download ZIP](https://github.com/yourname/liturgy-slide-remote/archive/main.zip)

## ▶️ How to Use
1. Extract the ZIP folder
2. Open your PowerPoint presentation
3. **Start slideshow (press F5)**
4. Double-click `start.bat`
5. On your phone, open Chrome/Firefox
6. scan
7. Tap buttons to switch slides

## 📱 Phone Interface
- ◀ PREVIOUS — Go back one slide
- NEXT ▶ — Advance one slide
- Jump to: [__] GO — Type slide number
- ▶ Start / ⏹ End — Start or end slideshow

> Tested on Windows 7, PowerPoint 2010, Android 8+
